# Easy Links

**[中文 | [English](README_EN.md)]**

易于使用的文件链接工具，通过可视化界面，全局快捷键，帮助您轻松创建文件/文件夹的符号链接与硬链接。

## 注意点

- 文件夹不支持创建硬链接，所以当你尝试对一个文件夹进行硬链接时，程序会遍历文件夹下所有文件，并以相同的目录结构硬链接每个文件至目标路径。
